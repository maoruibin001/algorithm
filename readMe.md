# js算法演练
* 各种排序算法的js实现。
* 对常用的排序提供优化方案。
* 提供单元测试。
## 项目目录
* +src 源代码目录。
* +test 单元测试目录。
## 注意事项
* js统一采用es6语法
* 测试使用mocha + Istanbul

## 运行测试
```
cd path to project
npm install 
npm run test
```
